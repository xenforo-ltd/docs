# Forum and thread types
The basic structure of a forum is a container for threads, those being a collection of posts, but from [version 2.2](https://xenforo.com/community/threads/forum-and-thread-types.181789/) onwards, XenForo allows that model to be extended to support a variety of different behaviours.

Forum and thread types include basic [discussions](discussions.mdx), which can contain [polls](discussions.mdx#poll-threads), [articles](articles.mdx) where the first post in a forum is made especially prominent, [questions](questions.mdx) which can have any number of answers and an accepted [solution](questions.mdx#solutions), and [suggestions](suggestions.mdx) where users can influence the popularity of an idea with votes.

## Caution when moving threads
Care must be taken when threads are moved between forums, to ensure that any special behaviours can be retained at their destination.

[A detailed discussion of the topic is available for your reference](thread-type-tools.mdx).
