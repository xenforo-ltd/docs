# Thread type tools

Additional attention must be paid to threads when operations are performed upon
them that would move them from one forum to another.

For example, moving a
[discussion thread](available/discussions.mdx#discussion-threads) into a
[suggestion forum](available/suggestions.mdx#suggestion-forums) (which can
contain only [suggestion threads](available/suggestions.mdx#suggestion-threads))
will automatically convert the thread type from
[discussion](available/discussions.mdx#discussion-threads) to
[suggestion](available/suggestions.mdx#suggestion-threads), and moving a
[suggestion thread](available/suggestions.mdx#suggestion-threads) into a
[discussion forum](available/discussions.mdx#discussion-forums) will convert it
to a plain [discussion](available/discussions.mdx#discussion-threads), which
will cause all votes cast for that suggestion to be lost.

:::warning Moving a [discussion](available/discussions.mdx#discussion-threads),
[article](available/articles.mdx#article-threads) or
[question](available/questions.mdx#question-threads) thread out of a dedicated
[discussion](available/discussions.mdx#discussion-forums),
[article](available/articles.mdx#article-forums) or
[question](available/questions.mdx#question-forums) forum into a
[mixed-type forum](available/discussions.mdx#mixed-type-forums) will allow the
thread type to remain, though of course it will lose the forum-specific
behaviours it had when it belonged to a type-specific forum. :::

Suggestion threads may **only** be moved into
[suggestion forums](available/suggestions.mdx#suggestion-forums) if the special
properties of suggestion threads are to be retained.

:::note These notes for _moving_ threads also apply to other operations that may
_effectively_ move threads from one forum to another, such as **merging**,
**copying** and **spam cleaning**. :::
