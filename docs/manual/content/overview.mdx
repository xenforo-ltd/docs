# User-generated content

*Content* describes all user-generated data throughout the XenForo system, including threads and posts created within forums. Much of this content shares various facilities for enhancement, including [tagging](tags.mdx), [smilies](smilies.mdx), [reactions](reactions.mdx), [voting](content-voting.mdx) and [BB code text markup](bbcode.mdx).

Additionally, some content types support embedding of [uploaded files and images (attachments)](attachments.mdx), [media hosted by external sources](bbcode-media-sites.mdx) such as online video, and GIF images from services such as [Giphy](giphy.mdx)
