# User-generated content

*Content* describes all user-generated data throughout the XenForo system, including threads and posts created within forums. Much of this content shares various facilities for enhancement, including [tagging](attributes/tags.mdx), [smilies](smilies.mdx), [reactions](attributes/reactions.mdx), [voting](attributes/content-voting.mdx) and [BB code text markup](bbcode.mdx).

Additionally, some content types support embedding of [uploaded files and images (attachments)](embeds/attachments.mdx), [media hosted by external sources](embeds/bbcode-media-sites.mdx) such as online video, and GIF images from services such as [Giphy](embeds/giphy.mdx)
